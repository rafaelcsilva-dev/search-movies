# 🎬 Seach Movie

Este projeto é uma aplicação web desenvolvida com **Next.js + React + TypeScript** que consome a API do The Movie Database (TMDB) para exibir uma lista de filmes populares.

---

# 📸 Preview

A aplicação mostra:

- Poster do filme
- Título
- Descrição
- Avaliação em estrelas

---

# 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido com as seguintes tecnologias:

- Next.js
- React
- TypeScript
- Axios
- CSS
- TMDB API (The Movie Database)

---

# 🌐 API Utilizada

A aplicação utiliza a API do **The Movie Database (TMDB)** para obter os dados dos filmes.
